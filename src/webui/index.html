L1 短期记忆（当前 session 推理步骤）
启用短期记忆 保留当前 session 最近 N 步的 Thought/Action/Observation 作为 scratchpad 注入 prompt。
启用步骤蒸馏 窗口溢出时用 LLM 压缩驱逐步骤，生成摘要注入当前 prompt，避免丢失推理上下文。
L2 中期记忆（近期对话历史）
启用近期历史 自动记录每轮对话的 Q&A，下次启动时加载最近 N 天的记录注入 system prompt。
启用滚动整合 超出最大条目数后，用 LLM 把最旧的一批条目蒸馏为摘要，每天最多自动整合一次。需 LLM 就绪。
L3 长期记忆（FAISS 向量库，需 faiss-cpu）
启用长期记忆 跨 session 向量化存储全部历史 Q&A，每轮根据问题语义检索相关记忆注入 prompt。需安装 faiss-cpu 和 sentence-transformers。
启用知识蒸馏 每轮结束后调用 LLM 将 Q&A 提炼为一条精炼知识摘要再写入向量库，减少噪声、提升检索质量。关闭时仅存原始答案文本。
里程碑记忆（LLM 重要性评分，稍慢）
启用里程碑 每轮对话结束后 LLM 评分，评分超过阈值的 Q&A 写入里程碑库，按关键词检索注入 prompt。
